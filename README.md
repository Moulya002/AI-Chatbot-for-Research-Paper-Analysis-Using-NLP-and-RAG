# AI-Powered Chatbot for Research Paper Analysis

## 📌 Project Overview
This project aims to build an intelligent chatbot using NLP and Retrieval-Augmented Generation (RAG) to analyze and retrieve insights from research papers.

## 📊 Dataset
- Source: ArXiv API
- Data includes titles, abstracts, categories

## 🔍 Project Phases
- Phase 1: Exploratory Data Analysis (EDA)
- Phase 2: Feature Engineering
- Phase 3: Model Optimization & Evaluation
- Phase 4: Deployment & Explainability (XAI)

## ⚙️ Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
